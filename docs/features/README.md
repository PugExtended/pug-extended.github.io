# Features

## Completions

Pug Extended Comes packed with a whole ton of completions such as:

- **Keywords** `eg. includes`
- **Tags** `eg. head(lang='en')`
- **Attributes** `eg. class-"class-name"`
- **Filters** `eg. :css(opts)`
